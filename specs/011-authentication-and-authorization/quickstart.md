# Quickstart: Authentication and Authorization

**Spec Number**: 011

## Setup

1. Review the spec: `specs/011-authentication-and-authorization/spec.md`
2. Review the plan: `plan.md`
3. Review tasks: `tasks.md`
4. Review tests: `tests.md`

## Development

1. Pick a task from `tasks.md`
2. Implement the feature
3. Write tests per `tests.md`
4. Run tests: `npm run test`
5. Mark task done: `/task-done <task-id>`

## Verification

1. Run `/scope-validate 011` to check spec quality
2. Run tests to verify implementation
3. Review test coverage against `tests.md` requirements
